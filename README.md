# Vape Shop - Telegram Mini App

Полнофункциональное веб-приложение для продажи вейпов с Telegram Mini App, админ-панелью и чатом поддержки.

## 🎨 Особенности

- **Дизайн Glassmorphism** - Современный и стильный интерфейс с эффектом стекла
- **Каталог товаров** - Фильтрация по категориям, быстрый поиск
- **Корзина** - Добавление/удаление товаров, управление количеством
- **Оформление заказа** - Полная форма с выбором способа доставки
- **Чат с менеджером** - Прямое общение в приложении
- **Админ-панель** - Управление товарами, чатами и пользователями
- **Система рейтинга пользователей** - Автоматические титулы для покупателей

## 📂 Структура файлов

```
vape-shop/
```

## 🚀 Быстрый старт

### 1. Клонируем репозиторий

```bash
git clone https://github.com/jejwicugy30-cmd/vape-shop.git
cd vape-shop
```

### 2. Развертываем на GitHub Pages

1. Убедитесь, что в репозитории есть все файлы
2. Перейдите в Settings → Pages
3. Выберите ветку `main` как источник
4. Приложение будет доступно по адресу: `https://jejwicugy30-cmd.github.io/vape-shop/index.html`

### 3. Запускаем Telegram бота

```bash
# Установим зависимости
pip install -r requirements.txt

# Отредактируем bot.py - вставим токен бота
# BOT_TOKEN = 'ВАШ_ТОКЕН_БОТА_ЗДЕСЬ'

# Запустим бота
python bot.py
```

## 🤖 Настройка Telegram бота

1. Создайте бота через [@BotFather](https://t.me/botfather) в Telegram
2. Получите токен бота
3. Вставьте токен в файл `bot.py`
4. Установите Webhook или используйте polling (как в примере)

### Команды бота

- `/start` - Начать работу, открыть меню
- `/help` - Справка по использованию
- `/orders` - Список заказов (только администратор)
- `/users` - Статистика пользователей (только администратор)

## 🎯 Как использовать приложение

### Для обычных пользователей

1. **Откройте бота** - найдите бота в Telegram и нажмите `/start`
2. **Откройте магазин** - нажмите кнопку "🛍️ Открыть магазин"
3. **Выбирайте товары** - используйте фильтры по категориям
4. **Добавляйте в корзину** - нажимайте "Добавить"
5. **Оформляйте заказ** - заполните форму и подтвердите
6. **Общайтесь с менеджером** - используйте вкладку "Чат"

### Для администратора

1. **ID администратора**: 7705124098
2. Откройте приложение - откроется админ-панель
3. Управляйте товарами, отвечайте на сообщения, отслеживайте пользователей

## 💾 Хранение данных

Приложение использует **LocalStorage** для хранения данных:

- `products` - Список товаров
- `cart_[userId]` - Корзина пользователя
- `chats` - Все чаты и сообщения
- `users` - Данные пользователей
- `orders` - История заказов

⚠️ **Важно**: LocalStorage очищается при очистке кэша браузера

## 🎨 Кастомизация

### Изменение цветов

Отредактируйте переменные в `styles.css`:

```css
:root {
    --primary: #8B5CF6;      /* Основной цвет */
    --secondary: #06B6D4;    /* Вторичный цвет */
    --accent: #EC4899;       /* Цвет акцента */
}
```

### Изменение товаров

В файле `app.js` найдите функцию `loadProducts()` и отредактируйте массив товаров.
